1. Update version in project -> properties -> AssemblyInfo.cs (bottom of the file AssemblyVersion & AssemblyFileVersion)
2. Open CMD and cd to project location
3. In cmd write nuget pack
4. Once successful it will create a 'SecureNative.SDK.[Version].nupkg' file that yo need to upload to nuget.org
5. Login to nuget.org with Nevo's account
6. Upload .nupkg file
7. Wait 10-60 minutes for Microsoft's vaidation to finish.


That's it.
